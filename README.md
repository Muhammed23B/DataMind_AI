# 🧠 DataMind AI – Natural Language to SQL Query System

DataMind AI is a Generative AI-powered database assistant that enables users to interact with databases using natural language. Instead of writing SQL queries manually, users can ask questions in plain English and receive accurate answers generated through AI-powered SQL query generation and execution.

The project combines Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), Semantic Similarity Search, and SQL databases to create an intelligent database question-answering system.

---

## 🚀 Features

- Natural Language to SQL Conversion
- AI-Powered Database Question Answering
- Semantic Similarity-Based Few-Shot Learning
- Automated SQL Query Generation
- Real-Time Query Execution
- Intelligent Response Generation
- Streamlit-Based User Interface
- Google Gemini Integration
- ChromaDB Vector Store for Example Retrieval

---

## 📌 Project Overview

DataMind AI allows users to ask business-related questions in natural language such as:

- "How many products were sold last month?"
- "Which category generated the highest revenue?"
- "Show total sales by region."
- "What is the average order value?"

The system automatically:

1. Understands the user's intent.
2. Retrieves similar examples using semantic search.
3. Generates optimized SQL queries using Gemini.
4. Executes the SQL query against the database.
5. Returns results in an easy-to-understand format.

## ⚙️ Workflow

### Step 1: User Enters a Question

Example:

```text
What is the total sales amount generated by Nike products?
```

### Step 2: Similar Examples Retrieved

Relevant examples are fetched from ChromaDB using semantic similarity search.

### Step 3: SQL Query Generation

Gemini generates an optimized SQL query.

Example:

```sql
SELECT SUM(price * quantity)
FROM sales
WHERE brand = 'Nike';
```

### Step 4: Query Execution

The generated query is executed against the MySQL database.

### Step 5: Response Generation

The results are returned to the user through Streamlit.

## 🔍 Key Components

### Few-Shot Learning

The application uses curated examples to improve SQL generation accuracy.

### Semantic Search

ChromaDB retrieves the most relevant examples based on the user's question.

### LangChain Pipeline

LangChain orchestrates:

- Prompt Engineering
- Example Selection
- LLM Interaction
- Query Generation

### Google Gemini

Gemini acts as the reasoning engine for converting natural language into SQL.

## ▶️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/DataMind_AI.git
cd DataMind_AI
```

### Create Virtual Environment

```bash
python -m venv venv
```

Activate environment:

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / Mac

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Environment Variables

Create a `.env` file:

```env
GOOGLE_API_KEY=your_gemini_api_key
DB_USER=your_database_user
DB_PASSWORD=your_database_password
DB_HOST=localhost
DB_NAME=your_database_name
```

### Run Application

```bash
streamlit run main.py
```

Application runs on:

```text
http://localhost:8501
```
